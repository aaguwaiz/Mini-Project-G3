Medical Expense Cost Prediction

Main Objective
The main objective of this project is to use existing medical expense for predicting future medical expenses of individuals to help medical insurance to make decision on charging the premium , using machine learning, and compare the performance of Support Vector Regression (SVR) against Linear Regression to find the most accurate model.

Dataset
The dataset contains historical insurance records with the following features:

Age

Sex

BMI

Children

Smoker

Region

Expenses (Target variable)

Models & Performance
We trained and evaluated two models on the dataset:

Support Vector Regression (SVR):

R2 Score: 88.40%

RMSE: 4616.27

Linear Regression:

R2 Score: 80.69%

RMSE: 5956.63

Conclusion
The SVR model outperformed Linear Regression, making it the selected model for this project due to its higher accuracy and lower prediction error.
